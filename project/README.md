# Lira Cosmetics AI - Backend

This folder contains all the Python code for the AI assistant and the cost tracking system.

## Main Files
-   `app.py`: The Flask server that talks to the frontend.
-   `chatbot.py`: Logic for sending messages to the AI APIs.
-   `cost_calculator.py`: The script that calculates how much each message costs.
-   `simulation.py`: A script I wrote to simulate many customers and gather data.

## Setup
1.  Set up your virtual environment.
2.  Install requirements.
3.  Add your API keys to the `.env` file.
4.  Run `python app.py`.
